# 🤖 VAN_AI - Day 1 (Basic Math Engine)

🚀 A powerful command-line based math engine built in Python using SymPy.  
This is **Day 1** of the VAN_AI project — focused on core mathematical operations.

---

# 📌 Features

## 🔢 Basic Arithmetic
- Addition, subtraction, multiplication, division  
- Power operations (`^`)  
- Square root (`sqrt()`)

## 📊 Factorial
- Supports `5!` and `(5)!`  
- Only valid for non-negative integers

## 🔢 Complex Numbers
- Supports `i` notation  
- Example:
  2+3i  
  (2+3i)*(1-i)

## 📊 Combinatorics
- Combination: `nCr(n, r)`  
- Permutation: `nPr(n, r)`

## 📐 Number Theory
- LCM: `lcm(12,18)`  
- HCF: `hcf(12,18)`

## 📦 Algebraic Factoring
- Factor expressions:
  factor(x^2 - 9)

---

# 🛠 Installation

1. Install Python  
2. Install SymPy:
   pip install sympy  

3. Run the program:
   python main.py

Start the program:

# ⚠️ Rules & Notes

- Factorial works only for non-negative integers  
- nCr, nPr, lcm, hcf require integers only  
- Use `i` for complex numbers  
- Division by zero is handled safely  
- Invalid input shows clean error messages
